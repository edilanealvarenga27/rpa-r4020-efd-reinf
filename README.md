# RPA R-4020 EFD-Reinf

## Sobre o Projeto

Este projeto consiste em uma solução de Robotic Process Automation (RPA) desenvolvida para automatizar tarefas operacionais relacionadas ao evento R-4020 da EFD-Reinf em contexto de Administração Pública.

A solução foi desenvolvida em Python utilizando automação de interface gráfica com PyAutoGUI, integração com planilhas Excel e interface gráfica amigável para parametrização dinâmica das execuções.

O robô permite:

- leitura automatizada de planilhas Excel;
- parametrização dinâmica de colunas e linhas;
- preenchimento automático de informações no sistema;
- geração de logs detalhados de execução;
- captura automática de erros;
- acompanhamento visual da automação em tempo real.

Os resultados obtidos demonstraram ganhos significativos de eficiência operacional, redução do tempo de execução das atividades e aumento da capacidade produtiva organizacional.

---

## Tecnologias Utilizadas

- Python 3.11
- PyAutoGUI
- OpenPyXL
- Tkinter
- PyInstaller
- Pyperclip

---

## Funcionalidades

- Interface gráfica amigável
- Captura dinâmica da posição do botão R-4020
- Configuração personalizada de colunas da planilha
- Logs detalhados em tempo real
- Botão de parada segura
- Captura automática de screenshots em erros
- Geração de executável standalone
